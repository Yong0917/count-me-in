# 설계: "오늘 응답 필요" 강조 표시

> 작성일: 2026-06-12

## 배경

홈 화면(달력 + 현황)에 진입했을 때, 오늘 일정에서 아직 참석 여부를 정하지 않은(미정) 멤버가
이를 놓치기 쉽다. PRD 성공 지표 중 "모임 전날 기준 미정 비율 ≤ 20%"에 기여하기 위해,
오늘 일정에서 본인이 미정 상태일 때 응답을 유도하는 시각적 강조를 추가한다.

## 범위

- 적용 대상: **오늘 일정만** (`occurrence.date === todayYmd()`).
- 적용 조건: 현재 멤버의 참석 상태(`myStatus`)가 `maybe`인 경우. (무응답·명시적 "미정" 모두 포함)
- 오늘이 아닌 날짜를 선택했을 때는 강조하지 않음.

## 표시 방식

`EventCard`의 "내 참석" 섹션에 강조 스타일을 적용한다.

- 색상: 기존 `--color-maybe` / `--color-maybe-soft`(앰버) 토큰 재사용 — 새 색상 토큰 불필요.
  "미정" 상태와 시각적으로 자연스럽게 연결된다.
- 안내 문구 추가: 예) "오늘 참석 여부를 알려주세요"
- 구체적 테두리/배경/패딩 값은 구현 시 기존 카드 디자인에 맞춰 조정.

## 변경 범위

- `src/components/EventCard.tsx` 1개 파일만 수정.
  - `todayYmd()`를 `@/lib/date`에서 import.
  - `const isToday = occurrence.date === todayYmd();`
  - `isToday && myStatus === "maybe"`일 때 "내 참석" 섹션에 강조 클래스 + 안내 문구 추가.
- 스키마/RPC/타입 변경 없음.

## 검증

별도 테스트 러너 없음 — 수동 확인:

1. 오늘 일정 + 본인 미정 상태 → "내 참석" 섹션 강조 표시됨.
2. 참석/불참으로 변경 → 강조 사라짐.
3. 오늘이 아닌 날짜의 일정 → 강조 없음.
4. 정기 일정의 가상(virtual) occurrence에서도 동일하게 동작.
